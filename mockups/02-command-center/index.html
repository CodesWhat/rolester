Capabilities

Every part of the loop,
built into the runtime.

Six skill families that cover the search cycle end to end — honest, evidence-grounded, and wired together so nothing falls through the cracks.

The Gate

Don't spray. Don't trust title/keyword matches. The gate reads the full posting body before tailoring. Emits a structured verdict against your actual constraints — comp floor, location, work mode, travel, work authorization, and honesty boundaries. Flags likely ghost jobs, evergreen reqs, and stale listings.

GATE KEEP
FIT 4.2 / 5
COMP ok — $170k base
LOCATION Remote ✓
TRAVEL ~20% — flag
ACTION Tailor + apply · flag travel at offer
Honest Tailoring

Artifacts built only from your evidence bank. A placeholder lint blocks unresolved tokens. No invented facts — if it isn't in your evidence, it doesn't go in the résumé.

✓ 0 placeholders  ·  ✓ 0 invented claims
Communication Memory

Recruiter threads, follow-ups, scheduling, and negotiation — all tracked. The agent drafts your replies from context, not a template, so every message fits the thread.

Interview Prep + Story Bank

Audience-segmented packets, STAR+result story bank keyed to the JD, verbal comp scripting, and live negotiation coaching — all from the same evidence you used to tailor.

Research Loop

Company intel, comp benchmarking, board discovery. A three-tier citation firewall keeps hallucinated "facts" out of your research output. Know what you're walking into.

Learning Memory

Per-role-family lessons compound across your search. What you learn from one cycle informs the next — gatekeeping, tailoring, and negotiation all get sharper over time.

Read-only dashboard

The agent does the work. The dashboard shows it.

A live local dashboard that renders your pipeline in real time as the agent works. Table, Board, and Calendar views. Five theme families. All read-only by design — mutations are the agent's job, not yours.

  • Active Pipeline with stage chips, fit scores, and company intel
  • Activity Pulse — live agent-feed showing what ran and when
  • Five theme families: Blue · Purple · Green · Orange · Dark
