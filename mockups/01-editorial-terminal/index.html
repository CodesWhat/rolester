What is Rolester

Rolester is a job-search workspace that runs entirely on your laptop. There's no cloud account, no subscription, and nothing that syncs your résumé or compensation numbers to a third party. You bring your own AI CLI, point it at the workspace, and the agent drives the work — reading full job postings, vetting them against your real constraints, building tailored artifacts only from your evidence, tracking every outcome, and helping you prep for each interview.

The same skills that serve a staff engineer serve a nurse or a logistics manager. Configuration, not code, sets the bias. The agent does the work; the dashboard shows the picture; you make the calls.

The Gate Rule

Don't tailor or apply until the job has passed a real body-read gate. Title and keyword matches are triage, not truth.

An opinionated body-read gate, not a keyword filter.

Reads the whole posting, not the title

The gate agent reads the full posting body before issuing any verdict. No triage on title or keyword. If the role doesn't clear your real constraints — comp floor, location, work mode, travel, work authorization — it doesn't pass.

Flags suspect listings, never silently drops them

Ghost postings, evergreen reqs, and stale listings get a flag — not an automatic reject. The gate surfaces the signal; you decide.

  • Posting age >60 days with no update
  • Likely evergreen / perpetual-fill pattern
  • Role title mismatch vs. posting body
  • Ghost posting signals (no named team, vague scope)

Emits a four-field verdict

GATE / FIT / COMP / ACTION. Everything downstream — tailoring, the cover letter, the tracker entry — only runs after a PASS verdict. No shortcuts.

Agent-agnostic

Bring your own AI CLI.

Rolester runs locally and uses an AI agent you already have — Claude Code, Codex CLI, or any CLI on your PATH — as its engine. It registers skills and data in a workspace; the AI agent reads and acts on them. Swap the agent without changing the skills or the dashboard.

The local web app opens automatically. The agent drives the work; the browser is where you watch it happen.

Honest tailoring

Built from your evidence, not invented

Every résumé, cover letter, and answer is assembled from your evidence bank — real work, real metrics, real claims. The workspace includes a placeholder lint step that blocks any document containing unresolved tokens. Rolester doesn't invent experience you don't have. It refuses to.

  • Evidence bank drives all output
  • Placeholder lint blocks unresolved tokens
  • No hallucinated credentials or metrics
  • You review before anything ships

Local & private

Your data stays on your machine

Rolester has zero runtime dependencies, no cloud account, and no telemetry. Your résumé, compensation numbers, and application pipeline live in a local workspace directory. Nothing is synced, transmitted, or stored anywhere else. You can inspect every file it touches.

  • Zero runtime dependencies
  • No account required
  • No telemetry, ever
  • Every file is human-readable on your disk

FAQ

Common questions.

Is my data really private?
Yes. Rolester runs entirely on your machine. There is no server, no sync, no account, and no telemetry. Your résumé, compensation floor, application pipeline, and all workspace data stay in a local directory that you control. The only time data leaves your machine is when your chosen AI CLI makes API calls to its own provider — that's between you and them, and you can inspect every prompt the agent sends.
Do I need to know how to code?
You need to be comfortable running a command in a terminal. That's genuinely about it. You'll type node bin/rolester.mjs start claude, and from there the agent asks questions to set up your workspace. You fill in YAML config files with your preferences and evidence — they're plain text, not code. Rolester is aimed at people who are comfortable with a CLI, but coding knowledge is not required.
Which AI CLIs work?
Claude Code (claude) and Codex CLI (codex) are tested. Any CLI that takes a prompt and operates on a filesystem should work. The skills are plain Markdown instruction files — there's nothing proprietary about the skill format. Pass start <your-cli-name> and Rolester hands off to it.
Does it apply for me automatically?
Not by default. Manual submission is the default and the recommended path — you review the tailored artifacts, then submit yourself. Auto-submit is opt-in and strictly bounded: the agent halts on any CAPTCHA or 2FA prompt, and it will not submit until a job has passed the full gate. There's no spray-and-pray mode. The point is to submit fewer, better applications, not more.
Is it free?
Rolester itself is free and open source, currently in pre-release. The only cost is whatever you pay for your AI CLI's API calls — that's billed directly by the provider you choose (Anthropic, OpenAI, etc.), not by Rolester. There is no Rolester subscription, no tier, no usage cap.
