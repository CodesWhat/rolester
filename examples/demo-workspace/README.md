# Demo Workspace

This directory is a tracked, fictional Rolester workspace fixture for demos,
website builds, screenshots, and docs.

It intentionally contains no real candidate data. The tracker data is copied from
the fictional demo pipeline, and the candidate config is copied from the sample
candidate fixture.

## Files

- `tracker.json` - fictional dashboard data.
- `candidate/*.yml` - fictional candidate config.

Use this directory as the stable public/demo data source. The live local
workspace remains `candidate/` and `workspace/`, both gitignored.
